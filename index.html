What networks and sub-networks exist in NYC's socialite community? Using data scraped from captions of newyorksocialdiary.com, I constructed a network of connections between attendees who are photographed together.

NBA statistics

Using the 'hidden' NBA stats API to visualize and analyze NBA data.

Shot Locations

A visualization of every shot attempted (27,768) in the 2015-16 NBA season as of November 18,2015. Shot logs provide x,y locations on the court for every shot attempted. This can be used to determine the most effective areas for each player as well as the best shooters in each region

Check it out here.

Player Tracking

The NBA tracks player and ball movement at a approximate 25Hz. This data can be acquired from the API and used to examine the saptiotemporal characteristics of plays. Since acquiring the data, I am able to parse and plot the data. The next step is to label the data manually as:

  • shot
  • pass
  • dribble
  • hold/possess

    • Then use that labeled data to build a model to automatically labels the data in other plays. This is useful to enhance the data set and get metrics for number of passes, time a player possesses the ball, and team spacing on offense and defense

    It's a work in progress, but have a look here.

    Predict Yelp Rating with only Business Attributes

    Supervised Machine Learning of Business Features

    What if I was thinking of opening a business and wanted to predict how well it would do before even opening? To do this I contructed a model using various features from yelp business parameters. You can see above the coefficient of determination from models contructed with each business feature.

    You can view the code and write up here.

    Predict Yelp Star Rating from Review Text

    Natural Language Processing, Supervised Learning

    In this project, I built several models to accurately predict the number of stars a user rates a business, given the review text. I used monograms, bigrams, and combinations of those features to accomplish this task.

    Jupyter notebookhere.

    Exploration of NYC Restaurant Inspection Data
